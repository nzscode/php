String Literals
  • Can use 'single quotes' or "double quotes"
  • strings can span multiple lines, the newline is part of the string
  • Single quotes strings can be used inside double quotes strings if they are escaped.
  • double quotes and \n can be used to expand a new line
  • single quotes with \n will not expand into a new line, nor will variables expand
  • Variables inside strings are expanded to a newline
